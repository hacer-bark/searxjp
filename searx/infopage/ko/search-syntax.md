# 검색 문법

SearXJP는 카테고리, 엔진, 언어 등을 수정할 수 있는 검색 문법을 제공합니다. 사용 가능한 엔진, 카테고리, 언어 목록은 환경설정에서 확인할 수 있습니다.

## `!` 엔진 및 카테고리 선택

카테고리나 엔진 이름을 설정하려면 `!` 접두사를 사용합니다. 몇 가지 예시는 다음과 같습니다:

- 위키백과에서 **서울** 검색:

  - {{search('!wp 서울')}}
  - {{search('!wikipedia 서울')}}

- **지도(map)** 카테고리에서 **서울** 검색:

  - {{search('!map 서울')}}

- 이미지 검색:

  - {{search('!images 한라산')}}

엔진과 언어의 약어도 사용할 수 있습니다. 엔진/카테고리 수식어는 연결하여 사용할 수 있으며 포괄적입니다. 예를 들어, {{search('!map !ddg !wp 서울')}}은 지도 카테고리에서 검색하고, 동시에 DuckDuckGo와 위키백과에서 **서울**을 검색합니다.

## `:` 언어 선택

언어 필터를 선택하려면 `:` 접두사를 사용합니다. 예시:

- 사용자 지정 언어(일본어)로 위키백과 검색:

  - {{search(':ja !wp 사무라이')}}

## `!!<bang>` 외부 Bang (바로가기)

SearXJP는 [DuckDuckGo]의 외부 Bang 기능을 지원합니다. 외부 검색 페이지로 바로 이동하려면 `!!` 접두사를 사용합니다. 예시:

- 위키백과(일본어판)에서 직접 검색:

  - {{search('!!wja 닌자')}}

이 검색은 외부 검색 엔진에서 직접 수행된다는 점에 유의하십시오. 이 경우 SearXJP는 귀하의 개인정보를 보호할 수 없습니다.

[DuckDuckGo]: https://duckduckgo.com/bang

## `!!` 자동 리다이렉트

검색 쿼리에 `!!`를 포함하면(공백으로 구분), 첫 번째 검색 결과로 자동 리다이렉트됩니다. 이 동작은 DuckDuckGo의 "I'm Feeling Lucky" 기능과 유사합니다. 예시:

- 쿼리를 검색하고 첫 번째 결과로 바로 이동

  - {{search('!! 일기예보')}}

리다이렉트되는 결과 사이트의 신뢰성은 검증할 수 없으며, 이 기능을 사용할 때 SearXJP는 귀하의 개인정보를 보호할 수 없음을 명심하십시오. 위험은 사용자가 감수해야 합니다.

## 특수 쿼리

환경설정 페이지에서 _특수 쿼리_ 키워드를 찾을 수 있습니다. 몇 가지 예시는 다음과 같습니다:

- 무작위 UUID 생성

  - {{search('random uuid')}}

- 평균값 계산

  - {{search('avg 123 548 2.04 24.2')}}

- 브라우저의 _User Agent_ 표시 (활성화 필요)

  - {{search('user-agent')}}

- 문자열을 다른 해시 다이제스트로 변환 (활성화 필요)

  - {{search('md5 lorem ipsum')}}
  - {{search('sha512 lorem ipsum')}}
