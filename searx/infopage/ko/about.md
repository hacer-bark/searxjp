# SearXJP 소개

SearXJP는 프라이버시가 강화되고 아시아 지역에 최적화된 [SearXNG] 메타 검색 엔진 인스턴스입니다.

일반적인 공개 인스턴스와 달리, SearXJP는 **「지식 분할(Split-Knowledge) 아키텍처」**를 기반으로 구축되었습니다. 우리는 일본 및 아시아 사용자를 위한 「저지연 액세스」와, 「누가 검색하는지 아는 서버」와 「무엇을 검색하는지 아는 서버」의 **물리적 분리**를 최우선으로 합니다.

## 우리의 아키텍처: 「지식 분할」 모델

SearXJP에서 귀하의 프라이버시를 침해하려면, 공격자는 서로 다른 사법권에 위치하고 서로 다른 제공자가 소유한 두 개의 서버를 동시에 침해해야 합니다.

우리는 엄격한 **「2노드 정책」**을 운영합니다:

### 1. 엔트리 노드 (일본 도쿄)
귀하가 연결하는 서버입니다. 이 서버는 엄격하게 **TCP 블라인드 리레이(TCP Blind Relay)**로만 작동합니다.
*   **아는 것:** 귀하의 IP 주소.
*   **모르는 것 (알 수 없는 것):** 귀하의 검색 쿼리.
*   **작동 방식:** 이 서버는 암호화된 SSL 패킷을 백엔드로 직접 전달합니다. SSL 키를 보유하지 않으므로, 이 서버가 귀하의 트래픽을 복호화하거나 읽는 것은 **수학적으로 불가능**합니다.

### 2. 처리 노드 (비공개 위치)
이 서버는 일본 엔트리 노드로부터 암호화된 전용 터널을 통해 트래픽을 수신합니다.
*   **아는 것:** 트래픽을 복호화하여 귀하가 무엇을 검색하는지 압니다.
*   **모르는 것 (알 수 없는 것):** 귀하의 IP 주소. 이 서버는 일본 엔트리 노드의 내부 IP만 볼 수 있습니다.
*   **작동 방식:** 이 노드는 연산 처리를 담당하며, Google/Bing/DuckDuckGo 등에서 결과를 집계하고 데이터를 익명화(Sanitize)한 후 터널을 통해 다시 전송합니다.

## 네트워크 성능 및 중국에서의 접근성

당사의 도쿄 엔트리 노드는 **GSL, PCCW, COGENT, EIE, JPNAP**를 포함한 프리미엄 BGP 네트워크 믹스를 사용합니다.

**중국 본토 사용자:**
중국 방화벽(Great Firewall)을 통한 접속 가능성을 공식적으로 테스트하지 않았으며 이를 보장할 수는 없지만, 당사의 네트워크 인프라는 국경 간 연결에 기술적으로 최적화되어 있습니다. **PCCW** 및 기타 프리미엄 아시아 경로가 포함되어 있어, 이론적으로 미국이나 유럽 인스턴스에 연결하는 것보다 우수한 지연 시간과 안정성을 제공합니다.

## 투명성: 로그 및 데이터 보존

우리는 무엇을 기록하고, 더 중요하게는 **「무엇을 기록할 수 없는지」**에 대해 급진적인 투명성을 믿습니다.

### 표준 인터넷 연결 (Clearweb)
*   **일본 엔트리 노드:** **최근 15분간의 TCP 연결 로그**를 유지합니다.
    *   *이유:* 이것은 전적으로 DDoS 방어와 속도 제한(Rate Limiting)을 위함입니다. 특정 IP가 과도한 연결을 시도하면 방화벽이 차단합니다.
    *   *프라이버시:* 이 노드는 암호화된 TCP 스트림만 처리하므로, **이 로그에 귀하의 검색 쿼리가 포함되는 것은 불가능합니다.**
*   **처리 노드 (백엔드):** **로그 없음 (No Logs).**
    *   IP 주소를 기록하지 않습니다 (볼 수 없습니다).
    *   검색 쿼리를 기록하지 않습니다.
    *   속도 제한은 사용자 IP가 아닌 「경로(Route)」(예: "글로벌 트래픽")를 기준으로 적용됩니다.

### Tor 및 I2P 연결
*   **엄격한 노로그 정책 (Strict No-Logs Policy):** Tor나 I2P를 통해 들어오는 트래픽은 표준 TCP 속도 제한 로그를 완전히 우회합니다.
*   익명 네트워크에 대한 진입 시간, 상관 데이터, 연결 메타데이터를 일체 추적하지 않습니다.

## 기본 검색 엔진으로 설정하는 방법

SearXJP는 [OpenSearch]를 지원합니다. 브라우저에 쉽게 추가할 수 있습니다:

- [Firefox] - 주소 표시줄을 마우스 오른쪽 버튼으로 클릭하고 "SearXJP 추가"를 선택하십시오.
- [Chromium] 기반 브라우저 (Chrome, Brave, Edge) - 설정 > 검색 엔진 > 검색 엔진 관리로 이동하여 자동으로 나타나지 않으면 SearXJP를 수동으로 추가하십시오.
